# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2026-01-07

### Added

- Added `LICENSE` file (MIT).
- Added `README.md` with features and usage instructions.
- Added `--version` flag to display the current version.
- Added `CHANGELOG.md` to track project changes.

### Changed

- Refactored project structure: moved headers to `include/space/`.
- Updated `CMakeLists.txt` to handle the new directory structure and install documentation.
- Removed emojis from `README.md` for a cleaner look.

## [1.0.0] - 2025-12-23

### Added

- Initial stable release.
- Core disk usage analysis functionality.
- ASCII bar visualization.
- Tree view with depth control.
- JSON output support.
- Arch Linux (PKGBUILD) support.
